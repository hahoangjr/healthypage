# Diet Goal Calculator

A simple static website for estimating BMI, maintenance calories, goal calories, and macros for:

- losing fat while maintaining muscle
- maintaining weight / recomposition
- gaining muscle / lean bulk

## How to publish with GitHub Pages

1. Create a GitHub repository.
2. Upload `index.html` to the repository.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. Wait a minute or two, then open the GitHub Pages URL.
